Pinewood is a typeface used mainly for display work. It's great in: ads, newsletters, headlines, or for any desktop publisher who wants that something ØextraÆ to stand out in their design work. Note: this typeface will use more memory than most, so don't download a lot of different fonts when you use this one or you'll overload your printer's memory. If you're using Adobe's Type Manager, you may tax it's memory use as well.  Remember, even if you use this font only once, it's well worth the price (See below).
