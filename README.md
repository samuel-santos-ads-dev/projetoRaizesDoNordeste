# Raízes do Nordeste

Projeto de front-end para uma rede de lanchonetes com autêntica comida nordestina. Desenvolvido como trabalho acadêmico para a UNINTER.

## Funcionalidades

*   Cadastro e login de usuários
*   Cardápio dinâmico por unidade (simulado com mock data)
*   Carrinho de compras com cálculo de total e previsão de pontos
*   Simulação de pagamento via Cartão ou Pix
*   Acompanhamento do status do pedido (Em preparo → Pronto → Retirado)
*   Programa de fidelidade com acúmulo e resgate de pontos
*   Conformidade com LGPD (Termos de Uso e Política de Privacidade)

## Tecnologias

*   HTML5 semântico
*   CSS3 (Mobile-first, Flexbox e Grid)
*   JavaScript (ES6+, localStorage, sessionStorage)
*   Dados mockados para simular o cardápio

## Como executar

1.  Clone este repositório.
2.  Abra o arquivo `index.html` em qualquer navegador.
3.  Para uma experiência completa, utilize o Live Server do VS Code.
